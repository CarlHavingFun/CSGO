# Generated Image Status

The images currently under this folder are early built-in `image_gen` drafts.

They are deprecated for final character use because:

- Faces are too homogeneous.
- Player archetype recognition is weak.
- Original player reference images were not used.
- ZywOo/Zyvo in particular does not match the required anchor: chubbier/rounder, glasses, curly hair.

Keep them only as examples of the rejected direction. Future accepted images should be made with Web ChatGPT using original reference images and side-by-side comparison.
